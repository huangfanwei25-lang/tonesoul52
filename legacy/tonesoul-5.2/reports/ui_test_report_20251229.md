### 遠端桌面問題詳解

**目前狀態：**
- 截圖會存成檔案 `runtime/screenshots/`
- 用戶需要手動查看檔案
- 沒有即時預覽

**用戶期望：**
- 網頁上直接顯示遠端桌面畫面
- 像 VNC / Remote Desktop 那樣即時

**技術限制：**
- Streamlit 不支援即時串流
- 需要 WebSocket 或 WebRTC

**可行方案：**
1. **定時截圖刷新**（簡單）：每 N 秒自動截圖並顯示在頁面上
2. **iframe 嵌入**（中等）：如果有其他遠端桌面服務
3. **WebRTC 串流**（複雜）：需要額外伺服器
