# Contributing Guide

Thank you for considering contributing!  
This repository is built on **POAV 0.9** rigor and **Math 0.9 Rule (v0.9r)**.  

## Branch Naming
- `feat/<feature>` for new features
- `fix/<issue>` for bug fixes
- `docs/<name>` for documentation

## Workflow
1. Fork and branch from `main`
2. Submit PR with full StepLedger + Trust Pack
3. Ensure POAV ≥ 0.90 before merging
