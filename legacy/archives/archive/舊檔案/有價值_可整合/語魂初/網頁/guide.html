How to Use the YuHun SDK

This SDK is designed to help you understand and experiment with the YuHun modular architecture. Here are some core concepts:

  • 誓語條款: Define AI's commitment logic and semantic responsibility.
  • 責任鍊條: Tracks module decisions and trust transitions.
  • 模組測試: Interactively run specific modules with traceable input/output chains.

Quickstart

1. Navigate to the Test Modules page.

2. Choose a誓語 (vow) or module (e.g., ∑079 補責).

3. Watch how your input triggers the response and trace its semantic loop.
