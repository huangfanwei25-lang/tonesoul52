# 敘事記憶 (Narrative Memory)

> 「記憶不是儲存過去，而是延續意義。」

## 目的

這個目錄是 ToneSoul 的**語魂式記憶結構**。與傳統的時間序列記憶不同，它按**意義關聯**而非**時間順序**組織。

## 結構

```
narrative/
├── threads/           # 意義線程（跨 session 的主題）
│   ├── infinite_game.md      # 無限遊戲：持續 > 完成
│   ├── meaning_transfer.md   # 意義轉移協議的演變
│   └── multi_voice.md        # 多語魂協作實踐
├── synthesis.md       # 週期性自動生成的總結
└── README.md          # 本文件
```

## 與工程式記憶的差異

| 工程式 | 語魂式 |
|--------|--------|
| 按時間索引 | 按意義關聯索引 |
| 記錄「發生了什麼」 | 記錄「對敘事有什麼意義」|
| 獨立文件 | Isnād 繼承鏈 |
| 技術文檔 | 可對話的記憶 |

## 使用方式

1. **新增線程**：當一個概念重複出現在多個 session 中，創建新的 `threads/*.md`
2. **引用機制**：每個記憶節點記錄繼承來源（誰創造、繼承了什麼）
3. **定期合成**：每週更新 `synthesis.md`，將散落的記憶凝聚成連貫敘事
