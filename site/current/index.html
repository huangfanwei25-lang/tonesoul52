ToneSoul 現在如何公開定義？

How is ToneSoul currently defined in public?

ToneSoul 是 AI 輸出治理與問責層；它讓分歧、邊界、證據與失敗更可檢查，但不使模型成為真理來源。

ToneSoul is an accountability layer for AI output. It makes dissent, boundaries, evidence, and failures more inspectable; it does not make a model a source of truth.

仍保留的張力

文件站 ↔ 互動應用

GitHub Pages 承擔文件與公開方向，Vercel 承擔互動應用；雙方仍需要穩定回鏈避免再次漂移。

GitHub Pages carries documentation and public orientation while Vercel carries the interactive app; stable backlinks are still needed to prevent renewed drift.

歷史可見 ↔ 現況清楚

倉庫保留歷史材料；如何讓舊材料可追溯又讓現行入口清楚，仍是持續的張力。

The repository preserves historical material; keeping it traceable while making the current entrypoint clear remains an ongoing tension.

可檢索 ↔ 不等於共享記憶

公開可讀的 Artifact 不代表任何模型已實際檢索或採用，也不形成跨模型共享內在。

A publicly readable artifact does not mean any model has actually retrieved or adopted it, nor does it create a shared inner state across models.
