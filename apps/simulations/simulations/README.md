# ToneSoul Simulations

This directory contains stress tests and simulation scenarios for the ToneSoul Architecture Engine.

## Purpose
- Validate system stability under high load.
- Test "Regret Reflex" and "Circuit Breaker" mechanisms.
- Simulate long-term memory drift.

## Usage
Run simulations via the main Makefile or directly via python:
```bash
python -m simulations.stress_test
```
