# 按鍵精靈快速啟動指南

## 🎮 功能說明
自動化按鍵工具，支援多種按鍵動作，每10分鐘自動執行一次。

## 📦 使用前準備

### 安裝依賴
```bash
pip install pyautogui
```

## 🚀 快速啟動

### 方法一：直接執行（推薦）
```bash
python key_macro.py
```

### 方法二：從 scripts 資料夾
```bash
python ../key_macro.py
```

## ⚙️ 新功能

### 支援的按鍵動作
1. **Alt+Enter** - 組合鍵（預設）
2. **Enter** - 單一按鍵（確認鍵）
3. **Space** - 空白鍵
4. **F5** - 重新整理

### 使用說明
1. 啟動程式後，從「按鍵動作」下拉選單選擇要執行的動作
2. 設定「間隔(分鐘)」（預設10分鐘）
3. 點擊「🧪 測試」可以先測試按鍵（3秒後執行）
4. 確認無誤後點擊「▶️ 開始」
5. 可隨時「⏸️ 暫停」或「⏹️ 停止」

## 💡 使用場景

### 場景1：自動確認對話
- 選擇：**Enter**
- 用途：每10分鐘自動按確認鍵

### 場景2：保持應用程式活躍
- 選擇：**Alt+Enter** 或 **Space**
- 用途：避免應用程式閒置

### 場景3：自動重新整理
- 選擇：**F5**
- 用途：網頁或應用程式定期重新整理

## ⚠️ 注意事項

1. **測試先行**：使用「🧪 測試」功能確認按鍵正確
2. **焦點視窗**：按鍵會發送到當前焦點視窗
3. **緊急停止**：移動滑鼠到螢幕角落可緊急停止（pyautogui 安全機制）
4. **最小化運行**：啟動後可將視窗最小化，不影響功能

## 🔧 技術資訊

- **引擎**: pyautogui (優先) / pynput (備用)
- **GUI**: tkinter
- **多執行緒**: 背景運行不阻塞介面
