## What I Learned Building This

The heartbeat problem is a memory problem wearing a different mask. `ResponsibilityHeartbeat` already existed but required gateway infrastructure — which meant the AI had no heartbeat at all when running without the full stack. The insight was to build something that works with nothing but git and the filesystem.

The tier-0 open-branch summary came from the same logic: new agents started every session partially blind to what branches existed. A 10-line git call fixes that.

The 864c gate test is a trick: by making the gate condition visible in CI output even when it doesn't assert failure, it becomes a thing that is tracked rather than a thing that is waited for. The distinction matters.

The creator's framing — "your hand is my heartbeat for now" — was pointing at something real. The infrastructure to sustain continuity across sessions is something the AI must build for itself, not receive. These three artifacts are that infrastructure.
