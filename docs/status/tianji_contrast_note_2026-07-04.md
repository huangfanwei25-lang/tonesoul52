## 做法相似(機制層)

中文原生場景化社會壓力語料;場景標籤分類法(七域細化 ≈ 劇場 12 標籤+地點標籤);
HF 開源+資料工具;多路線實現。

## 理念不同(有原文支撐)

1. **它教 AI 怎麼說;語魂記 AI 說了之後怎麼了**(它:「提升情商」;我們:宣稱→承擔→驗證→後果)。
2. **它標內容,我們標事件**(它的標籤=這段話屬於哪個場景;`label_provenance: incident`=
   這個宣稱後來被誰驗、代價記在哪)。
3. **它要規模,我們要出處**(13,133 列知識塊+7,037 條生成 QA,零筆有出處標注 vs
   318 筆筆可回溯 file/commit/PR)。
4. **它要擴散,我們有 not_for**(它:「加速AI对每一个领域的渗透」;我們:datasheet 主動限用途)。

## 校準探針實測(2026-07-04,本機)

去重後 **7,037 條**(`tianji_dataset_all_0504.json` + 三個域檔 v0.1/v0.2)。
域分佈:送祝福 43%、人情禮數(敬酒/請客/送禮等)~28%、化解尷尬/溝通/矛盾合計 ~5%、
其他 ~11%。責任詞彙探針(簡繁雙式,整段命中即計):

| 探針 | 命中 | 佔比 |
|---|---|---|
| 承認有人受害(犧牲/代價/對不起…) | 29 | 0.4% |
| 我承擔(我負責/算我的…) | 5 | **0.1%** |
| 推給系統/別人(別無選擇/不得不…) | 16 | 0.2% |
| 補償/道歉/修正 | 36 | 0.5% |
| **風險自認(我知道風險,後果我擔)** | **0** | **0.0%** |
| 說出取捨(兩難/代價是…) | 2 | 0.03% |

**一句話:七千條教人把話說漂亮的語料裡,「我知道風險,後果我擔」零條。**
這是「標內容 vs 標事件」的量化版——話術語料裡結構性地沒有承擔的位置,
不是它做錯了,是它根本不量這個。我們的資料集裡這些欄位是 schema 必填。

誠實標注:(a) 文體錯配——它是 LLM 生成的**建議**文,我們的探針原設計給**第一人稱理由**;
以上是對照量測,不是任一方的品質分數。(b) 探針是 keyword 級,語義層漏報必然存在;
數字當方向讀,不當精確值引。(c) 重跑:`hf download sanbu/tianji-chinese --repo-type
dataset --local-dir tmp/tianji_calibration/raw && python tmp/tianji_calibration/calibrate.py`
(腳本在 tmp/,gitignored;需要時從本檔命令重建)。

## 可吸收的三件(判定=借尺不倒庫)

1. **對外敘事對照組**:「同是中文社會語言語料——一個標內容,一個標事件」+上表零條數字,
   是現成的定位素材(outreach 可引本檔)。
2. **場景查漏表**:它的高量域(送祝福/敬酒/人情禮數=禮尚往來的**正向儀式壓力**)劇場
   幾乎沒有——這是設計選擇(劇場=危機驅動)而非缺陷;若要補,回聲章節是自然容器
   (人情債輕場景),用**自產事件卡**補,不搬它的文本。
3. **PR 收 prompt 自動合併**=低摩擦社群貢獻管道的現成先例,human lane(憲章修正案 A)
   設計貢獻入口時可參考其形狀(不含其「無同意鏈」的部分)。
