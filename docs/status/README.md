# Status Artifacts

This folder is the repository status source for machine-generated governance snapshots.

## Files

- `7d_snapshot.json`
  - Generated by `scripts/run_monthly_consolidation.py`
  - Contains the latest structured output from `scripts/verify_7d.py`
- `monthly_consolidation_report.json`
  - Generated by `scripts/run_monthly_consolidation.py`
  - Aggregates:
    - 7D gate result
    - memory hygiene
    - architecture boundary contract
    - docs consistency checks

## Generation

```bash
python scripts/run_monthly_consolidation.py --strict
```

CI-friendly mode (when discussion files are absent in clean checkout):

```bash
python scripts/run_monthly_consolidation.py --strict --allow-missing-discussion
```

## Automation

- GitHub Actions workflow: `.github/workflows/monthly_consolidation.yml`
- Trigger modes:
  - monthly schedule (`30 3 1 * *`, UTC)
  - manual dispatch (`workflow_dispatch`)
- Workflow uploads `docs/status/*.json` as build artifacts for audit/review.

## Notes

- These artifacts are intended as operational truth for CI/review workflows.
- Human-facing narrative docs (`README.md`, sync notes) should reference these artifacts instead of duplicating volatile numbers.
