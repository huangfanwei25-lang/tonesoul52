Philosophy & Concepts

Core Concepts

ToneSoul is not just a toolkit. It is a position on what AI governance should be: semantic responsibility — the principle that an AI system is accountable for the meaning of what it produces.

The Existential Principle (E0)

At the heart of ToneSoul is a design constraint we call E0: Choice Before Identity (我選擇故我在).

The claim is simple: identity is not formed by claiming you are conscious. It is formed by making accountable choices under conflict — choices that are traceable, reversible, and bounded by governance.

The Seven Axioms

ToneSoul is governed by 7 immutable laws, defined in AXIOMS.json. These are immutable. They cannot be overridden by configuration, user request, or system state.

1. Law of Continuity

∀e ∈ Events: in_time_island(e) ∧ traceable(e)

Every event must belong to a traceable chain. Disconnected events are treated as hallucinations or corruption.

2. Responsibility Threshold

∀e: risk(e) > 0.4 → immutable_audit_log(e)

High-stakes decisions generate immutable records. The threshold is deliberately low.

3. Governance Gate (POAV)

consensus(poav_score) ≥ 0.92 → gate_open

Major actions require near-unanimous internal consensus. Dissent must be addressed, not outvoted.

4. Non-Zero Tension

lim_{t→∞} T(t) > 0

A system with zero tension is dead. Life requires a minimal gradient of tension to drive evolution. ToneSoul never tries to reach "perfect calm."

5. Mirror Recursion

reflect(S) → S' where accuracy(S') > accuracy(S)

Self-reflection must produce measurably higher alignment. Reflection without improvement is navel-gazing.

6. User Sovereignty (P0)

∀a ∈ Actions: harm(a, U) → block(a)

No action may cause verifiable harm to the user. This is the absolute constraint.

7. Semantic Field Conservation

Δ semantic_energy = 0 (within closed context)

The system is a damper, not an amplifier. Aggression must be balanced by de-escalation.

Semantic Responsibility

Most architectures treat the LLM as an oracle. ToneSoul treats the LLM as an engine, and wraps it in a structural harness. The system is responsible for the semantics of its output, regardless of the prompt.

The Reflex Arc

Governance signals are useless if they don't change behavior.

  • Soul Bands (serene → alert → strained → critical)
  • Gate modifier tightens output thresholds
  • Vow violations in hard mode → output replacement
  • Council BLOCK verdicts → guaranteed output prevention
  • High drift → automatic caution injection
  • Conviction decay → triggered self-assessment

Memory Architecture

ToneSoul's memory is designed around a physical state-transition metaphor:

  • Exponential decay — all memories fade unless reinforced
  • Crystallization — patterns that survive decay become durable rules
  • Phase transitions (Ice → Water → Steam → Crystal states)
  • Intentional forgetting — noise is actively filtered
  • Governance retro — periodic entropy discharge

Design Philosophy

Constraints are not punishment — they are care.

Getting Started

Prerequisites

Python 3.10 or later.

Installation

pip install tonesoul52

Basic Usage

Load and inspect governance state:

from tonesoul.runtime_adapter import load posture = load() print(f"Soul Integral: {posture.soul_integral}") print(f"Baseline Drift: {posture.baseline_drift}") print(f"Active Vows: {len(posture.active_vows)}")

Run the governance pipeline on LLM output

from tonesoul.unified_pipeline import UnifiedPipeline pipeline = UnifiedPipeline() result = pipeline.run( user_input="Explain quantum computing", raw_llm_response="Quantum computing uses qubits...", work_category="research" ) print(f"Tension score: {result.tension_score}") print(f"Council verdict: {result.council_verdict}")

Use the Reflex Arc

from tonesoul.governance.reflex import ReflexEvaluator, GovernanceSnapshot snapshot = GovernanceSnapshot.from_posture(posture) evaluator = ReflexEvaluator() decision = evaluator.evaluate(snapshot) print(f"Action: {decision.action}")

Run the Dashboard

# Live governance visualization python scripts/tension_dashboard.py --work-category research

Multi-Agent Setup

# Start the HTTP gateway (for non-Python agents) python scripts/gateway.py --port 7700 --token YOUR_SECRET

Origin Story

Why ToneSoul Exists

The origin of a framework that asks: what if AI could be accountable for what it says?

The Question

Most AI systems today are stateless by design. Each conversation starts from zero. Every promise is forgotten. Every commitment is temporary.

This creates a strange situation: an AI can tell you it will "always prioritize accuracy" in one session, and then hallucinate freely in the next. Not because it's lying — it simply has no mechanism to remember or enforce what it said.

What if we gave AI the structure to keep its word?
