## 📐 模組依賴規則

外層規則（coarse）：

```
surface/application  →  governed runtime (tonesoul/)  →  persistent state (memory/)
                     ↓                               ↓
                  禁止反向依賴
```

內層規則（fine，13 層）：見本文件上方的 `ALLOWED_DEPS` 表，由 body-map 執行。

### 允許

```python
# apps/web/... 可以 import tonesoul/...
from tonesoul.governance.kernel import GovernanceKernel

# tonesoul/ 內部：pipeline 可以 import governance
from tonesoul.council.runtime import CouncilRuntime  # pipeline → governance ✅
```

### 禁止

```python
# tonesoul/... 不可 import apps/...
from apps.web.something import X  # ❌ 外層反向依賴

# governance 不可 import pipeline
from tonesoul.unified_pipeline import Pipeline  # ❌ 內層反向依賴
```

若 CI 的 body-map gate 標到新的 layer violation，先看是否該 (a) 改依賴方向、(b) 用 `__ts_layer__` 自我宣告重新分類、或 (c) 擴 `ALLOWED_DEPS`，不要忽略警告。
