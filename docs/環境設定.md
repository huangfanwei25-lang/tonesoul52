## 常見錯誤

1. `Backend unavailable` / `502`
   - 檢查 Vercel `TONESOUL_BACKEND_URL` 是否為完整 HTTPS URL。
   - 禁止使用 `mock`、`localhost`、`127.0.0.1`。
2. `Backend configuration invalid for Vercel runtime` / `503`
   - 表示環境變數值缺失、格式錯誤或非 HTTPS。
3. 可開首頁但無法對話
   - 檢查 Render 日誌、CORS、以及 `GEMINI_API_KEY` 是否設定。
