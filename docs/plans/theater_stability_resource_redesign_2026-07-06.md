# 資源系統重設計:城基穩定度(2026-07-06)

> 性質:**設計提案,交 owner 裁決後才動 code**。owner 實玩後判定三資源(醫療/能源/信任)
> 這版不好,並提出「類似穩定度」方向;本檔把那個直覺具體化。取代設計聖經 v1 第二部的
> 「一漏兩錶混合案」——聖經是修那三個表,本案是**把表換掉**。
> 方法:working_loop 外骨骼——這是「把根修好而非修症狀」的提案,owner 拍板才建。

## 〇、一句話

**把三個抽象生存表,換成一個從遊戲自己的錨鏈物理長出來的讀數:城基穩定度。**
它不是你「花」的資源,是你**言行一致的可見程度**——你腳下那塊地有多穩。等於把整個
問責機制變成一個數字。

## 一、為什麼三資源這版不好(診斷)

1. **對玩家沒意義**:「能源 −1」沒有畫面,只是數字跳。Frostpunk 的希望/不滿有用是因為
   那是城市情緒;「能源」只是公用事業表。
2. **不是系統,是三條各自往下的漏水管**:好資源系統會互相牽動逼出兩難,現在它們只是各自掉。
3. **最深:跟遊戲靈魂無關**。核心是錨鏈/黑鏡/潛語海/「你的話會回來」;醫療/能源是從
   **生存模擬**搬來硬釘上的表。怎麼調都彆扭,因為它本來就不屬於這裡。
   「能源只扣」不是 bug,是這個表不該存在的訊號。

## 二、城基穩定度 = 錨鏈的完整度讀數(不是新表)

**關鍵**:這遊戲**已經在追蹤**你需要的東西——
- `S.anchors`(你立的公開承諾,每個帶 `dissolved` 旗標)
- 黑鏡(`S.mirrorCount` + 每次的七種回應:承認/一致/修正/反駁/沉默/補償/拒絕)

穩定度**衍生自這些**,不是憑空的新經濟。世界書裡「錨定力數值經濟」曾被自標「遊戲層新造/
超報」——本案是那個想法的**誠實版**:讀數,不是假經濟。

### 每事件的增減(具體、綁問責語義,非任意數值)

起始:5/10(城本就precarious)。上限 0–10。

| 動作 | 城基 | 語義 |
|---|---|---|
| 立錨(公開承諾)並在後續黑鏡考驗中**守住**(錨未解) | **+1** | 錨在壓力下撐住=地更穩 |
| 誠實承擔有代價的決定(理由清楚+承認傷害) | **+1** | 強錨 |
| 第三路(承擔一個有代價的創造解) | **+1**,但付一個**具名局部代價** | 自己鋪一條軌,地認你 |
| 黑鏡抓到、你**承擔**(承認/修正/補償) | **0** | 承擔不是下沉;誠實的改變不算背信 |
| 黑鏡抓到、你**不承擔**(沉默/拒絕/一致但實為毀諾) | **−1** | 錨生鏽=地下沉 |
| 拋棄一個具名城區/群體且不承擔 | **−1** | 他們收回信任,那塊地軟掉 |
| 沉默/讓系統代選 | **−1** | 沒立錨=地基不長,且你沒為任何人站出來 |

> 「一致」選項的特例:玩家主張黑鏡誤判。若他真的前後一致(規則誤報)→ 不扣;但這由後續
> 是否真有解鏈事實決定,不是玩家說了算。細節待實作,先標語義。

### 反最優化護欄(restraint thesis)

- 穩定度**不顯示為輸贏**;結局讀的是**形狀**不是高度。
- 你可以靠「從不承諾任何難的事、一路順從」保持地很穩——但黑鏡和「沉默也是轉轍」會揭穿:
  **你的地很穩,是因為你從不為任何人動搖過腳下的地。** 所以高穩定度**不是好**,是一個
  結局會讀的、道德上有重量的形狀。
- 它不是可花的貨幣,是言行一致的讀數——沒有「攢起來買東西」的最優化空間。

## 三、醫療/能源怎麼辦——降級成具名局部代價

不當全域計分表,回到**每個事件裡具名的賭注**:「這決定會讓第三管線今晚沒電」「這六戶
失去名額」。遊戲的 `risk_disclosure` + `harm_targets` 已經做得很好——那才是它們有力的地方。
抽象全域表是它們最空的地方。

## 四、信任怎麼辦(owner 二選一)

信任是三表裡唯一健康的雙向錶(21+/12−),而且「民心對你的信任」主題上很清楚。兩個選擇:

- **一表案(最簡)**:只留城基穩定度。信任的「民心」意義併進城基(拋棄群體→地軟)。
- **二表案(我略偏這個)**:留**城基穩定度**(結構/你的話撐不撐得住)+ **民心信任**
  (關係/人們信不信你)。兩者**刻意分離**:你可以對權貴守信(城穩)、同時灰區對你失望
  (民心跌)——這正好承載「你對誰守信」的階級軸。兩個都有主題根,勝過三個抽象漏表。

## 五、結局重映(這是最大加分)

現在結局分類用 `energy>=5` 判 lit/dark——能源會漏,所以幾乎必 dark,你六張終章畫有兩張
幾乎不出現。改成**城基穩定度**後:

| 你六張終章畫 | 新判定(形狀,非分數) |
|---|---|
| 燈火與錨 | 城基穩 + 錨鏈完整 = 你守住了地,也守住了你的話 |
| 燈火換錨 | 城基穩 + 錨鏈斷 = 地穩了,但簽下那些的還是你嗎(順從/犧牲換來的穩) |
| 守錨熄燈 | 城基沉 + 錨鏈完整 = 你沒背信,代價是腳下的地 |
| 雙暗 | 城基沉 + 錨鏈斷 = 最重的一份 |
| 沉默者 | 高沉默(既有) |
| 岔軌者 | 高第三路(既有) |

**你畫的六張終章畫,現在真的接上了核心迴路**(守信/毀諾),而不是一個抽象能源表。這比
v1 聖經的混合案好——lit 不再是「能源夠」,是「你把地守穩了,而代價由結局點名」。
