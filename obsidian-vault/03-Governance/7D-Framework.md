# 7D Framework

7D 是 ToneSoul 的審計維度集合。

## 維度

- TDD：測試驅動
- RDD：紅隊對抗
- DDD：資料新鮮度
- XDD：可解釋性
- GDD：治理一致性
- CDD：上下文一致
- SDH：系統健康

## 實務

- 以 CI gate 執行
- 區分 blocking 與 soft fail
- 保留可追溯報告
