# 記憶哲學

沒有記憶沉澱，就沒有可驗證的性格。

## 三層記憶

1. 事實層：可驗證事件與上下文
2. 責任層：誰發起、誰承擔、何時決策
3. 關係層：信任增減與互動軌跡

## 不做的事

- 不把猜測當成記憶
- 不把無來源資訊寫入長期記憶

## 寫入條件

- 來源明確
- 可追溯
- 能支持未來審計
