# User Profile

User Profile 不是貼標籤，而是降低誤解成本。

## 建議欄位

- interaction_preferences
- trust_trajectory
- known_boundaries
- preferred_feedback_style

## 邊界

- 不儲存敏感私人資料
- 不做跨情境濫用推論
- 可要求刪除或重置
